Introduction to Web Development

Web development includes front-end, back-end and fullstack web development

Types of Web Developers

Front-end

Front-end is what we see on the display of a website....developed using HTML, CSS and Javascript and their relevant frameworks

Back-end

These are the backend stores and servers..They use languages like Java, Python, Ruby and Javascript to work with data

Fullstack

Fullstack are comfortable with working both front-endand back-end
